## What is StatHub?

StatHub is an open-source platform that lets you create and manage sports matches, track live scores, and stream real-time text commentary to connected clients over WebSockets.

It is built as a full-stack application:

- **Server** — Node.js + Express 5 REST API with a WebSocket server for real-time push
- **Database** — PostgreSQL managed through Drizzle ORM with type-safe schema definitions
- **Client** — React 19 single-page application with Tailwind CSS
- **Auth** — JWT-based sessions, Google OAuth, and email-based password reset

## Architecture overview

```
Browser (React) ──HTTP──▶ Express API ──▶ PostgreSQL
                └──WS───▶ WebSocket Server
```

The Express server and WebSocket server share the same HTTP port. When a match is created or commentary is posted via the REST API, the relevant broadcast function is called and all subscribed WebSocket clients receive the event immediately.

<Info>
  StatHub is self-hosted. You bring your own PostgreSQL instance and deploy the server wherever Node.js runs.
</Info>
